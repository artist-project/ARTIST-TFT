
Technical Feasibility Tool - TFT, Version 1.0.0

=================================

ABOUT
-----

Technical Feasibility Tool - TFT analyzes the component model of the cloud migration
candidate application and Maturity Assessment Tool's report and suggests migration 
strategies for each component. The tool also estimates the migration efforts and complexities 
with the complexity information gathered from TFT-SCC.

=========================================================================================

INCLUDED FUNCTIONALITIES
------------------------

- An Eclipse view visualizing the Maturity Assessment Tool Report which allows modifying the
target cloud platform. 
- An Eclipse view which which is responsible for analyzing the inputs (MAT report, component model,
SCC results) and showing the user the outcome of this analysis i.e. the migration strategies and
migration effort estimations. 
- An Eclipse dialog which allows the user to modify the suggested migration strategies
- Report generation
